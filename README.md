# SweetEdition Theme Engine

SweetEdition is a neon‑purple, glossy, KDE‑inspired theme engine designed to bring a unified Sweet aesthetic across your entire Linux environment.  
It includes GTK themes, Plasma styling, configuration modules, and a built‑in update system powered by GitHub Pages.

---

## ✨ Features

- **Unified Sweet Aesthetic**  
  Glossy neon‑purple UI elements inspired by KDE’s Sweet theme family.

- **GTK + Plasma Support**  
  Includes Sweet GTK variants and Plasma styling modules.

- **Auto‑Update System**  
  The web UI checks for new releases using `update.json` and displays an update banner when a new version is available.

- **Installer Scripts**  
  Includes `sweet-installer.sh` and an apply script for quick deployment.

- **Web‑Based Control Panel**  
  Hosted via GitHub Pages at:  
  **https://jim71g.github.io/SweetEdition**
